# 🎰 アナスロ出玉データ収集・分析システム (anaslo-scraper-ja)

本システムは、パチスロデータ分析サイト「アナスロ」から愛媛県松山市の店舗（初期設定：「大盛空港通り店」）のデータを自動収集し、SQLiteデータベースに保存した上で、StreamlitによるWeb画面で曜日別・末尾日別・台末尾別の出玉傾向を分析・可視化するローカルアプリケーションです。

---

## 1. 準備：Python環境のセットアップ (初回のみ)

本システムを動かすには、お使いのWindows PCに **Python 3.10以上** をインストールする必要があります。

### Pythonのインストール手順
1.  Python公式サイトのダウンロードページ（[python.org](https://www.python.org/downloads/)）にアクセスし、最新のWindows用インストーラーをダウンロードします。
2.  ダウンロードしたインストーラーを実行します。
3.  **【重要】** インストール画面の下部にある **`Add python.exe to PATH`**（または `Add Python to environment variables`）のチェックボックスに**必ずチェックを入れてから**、`Install Now` を選択してください。
4.  インストール完了後、PCを一度再起動します。

### 依存ライブラリのインストール
コマンドプロンプトまたはPowerShellを開き、本ツールのディレクトリに移動して以下のコマンドを実行します。

```bash
cd C:\Users\masat\Documents\GitHub\cc-sdd\tools\anaslo-scraper
pip install -r requirements.txt
```

## 2. システムの使い方

本システムは、対話式のメニュー画面またはコマンドライン引数を使って起動できます。

### 対話式メニューの起動
コマンドプロンプトで以下のコマンドを実行します。

```bash
cd C:\Users\masat\Documents\GitHub\cc-sdd\tools\anaslo-scraper
python run.py
```

実行すると、以下のようなメニューが表示されます。キーボードの番号 [1-5] を入力して操作します。

```
=========================================
   🎰 アナスロ出玉データ収集・分析システム
=========================================
 1. 最新データの自動収集 (未登録分のみ)
 2. 過去データの一括収集 (バックフィル)
 3. 指定日のデータ上書き (リフレッシュ)
 4. 分析ダッシュボードの起動 (Streamlit)
 5. 終了
=========================================
```

## 3. コマンドラインによる個別起動 (自動化用)

タスクスケジューラ等に登録する際は、以下のコマンドライン引数を指定して直接起動できます。

*   **最新データの自動収集 (未登録分のみ)**
    ```bash
    python run.py --collect
    ```
    ※起動時に自動でデータベースのバックアップ（最大5世代）が `backup/` フォルダに作成されます。

*   **過去データの一括収集 (バックフィル)**
    （例：過去30日分のデータを一括収集する場合）
    ```bash
    python run.py --backfill 30
    ```

*   **指定した日付のデータを再取得・上書き (リフレッシュ)**
    （例：2026年6月15日のデータを再取得する場合）
    ```bash
    python run.py --refresh 2026-06-15
    ```

*   **ダッシュボードのみを起動**
    ```bash
    python run.py --dashboard
    ```

## 4. Windows タスクスケジューラによるデータ収集の自動化

毎日決まった時間（例：データが更新される深夜AM 3:00など）に自動でデータを収集させる手順です。

1.  スタートメニューで「タスク スケジューラ」と検索して起動します。
2.  右側の「操作」パネルから「**基本タスクの作成**」を選択します。
3.  **名前**: 「Anaslo-Data-Collect」などと入力し、「次へ」を押します。
4.  **トリガー**: 「毎日」を選択し、任意の時間（例: `03:00:00`）を設定します。
5.  **操作**: 「プログラムの開始」を選択します。
6.  **プログラム/スクリプト**: `python`（またはPythonのフルパス `C:\Users\<ユーザー名>\AppData\Local\Programs\Python\Python312\python.exe`）を入力します。
7.  **引数の追加 (オプション)**: `run.py --collect`
8.  **開始 (オプション)**: `C:\Users\masat\Documents\GitHub\cc-sdd\tools\anaslo-scraper` (ツールのあるフォルダの絶対パス)
9.  「完了」を押して保存します。

## 5. ファイル構成とログ

*   `anaslo_data.db`: 収集されたデータがすべて格納されるSQLiteデータベースファイルです。
*   `logs/anaslo.log`: クローラーの実行ログやエラーが出力されます。データがうまく取得できない場合はこのログを確認してください（最大10MBで自動的にローテーション保存されます）。
*   `backup/`: 自動クローリング起動時にバックアップされたデータベースが最大5世代保存されます。
